# vinntravel-frontend
Vinn Travel frontend application

# Requirements
 Git -> you can download from  [https://git-scm.com/downloads](https://git-scm.com/downloads)

 Nodejs -> you can download from [https://nodejs.org/en/download/](https://nodejs.org/en/download/)

# Getting started
run following code to clone project

`git clone git@github.com:istanbulpatent/vinntravel-frontend.git`

then go to project folder

`cd vinntravel-frontend`

then change branch to dev branch

`git checkout dev`

run following command to install npm packages

`npm install`

run following command to start server

`npm start -s`

open browser and visit [http://localhost:3000/](http://localhost:3000/)
